# Commands

## Start a local tunnel

```
Cloudflare Tunnel: Start
```

## Stop your local tunnel

```
Cloudflare Tunnel: Stop
```

## Get the cloudflared version

```
Cloudflare Tunnel: Version
```

## Check if the tunnel is running

```
Cloudflare Tunnel: Is running?
```
